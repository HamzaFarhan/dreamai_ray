DF Processors

Functions to be applied on DataFrames for indexing and searching.

source

df_index_search

 df_index_search (df, ems_col='embedding', ems_key='embedding',
                  index_col='index', k=1, verbose=False)

Find the nearest neighbors of the embeddings in df.

Type Default Details
df
ems_col str embedding Column name of embeddings in df.
ems_key str embedding Key name of embeddings in json file.
index_col str index Column name of index in df.
k int 1 Number of nearest neighbors to return.
verbose bool False Whether to print out information.

df_to_index

 df_to_index (df, index, ems_col='embedding', ems_key='embedding',
              verbose=False)

Add the embeddings in df to the index.

Type Default Details
df
index
ems_col str embedding Column name of embeddings in df.
ems_key str embedding Key name of embeddings in json file.
verbose bool False Whether to print out information.

Usage Example

index_dim = 768
np.random.seed(42)
num_ems = 5
ems = [np.random.random((1, index_dim))[0].tolist() for i in range(num_ems)]
df = pd.DataFrame({"embedding": ems})
df.index = [i for i in range(10, 10 + num_ems)]  # Create a fake index.
index = create_index(index_dim)

df = df.apply(lambda x: df_to_index(x, index, verbose=True), axis=1)
faiss.write_index(index, "index.faiss")
