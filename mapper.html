Mapper

 Mapper (udf=<function noop>, udf_kwargs={}, cbs=[], verbose=False,
         **kwargs)

A class to map a function to a dataframe. The function should expect a dataframe as input and return a dataframe as output.

Type Default Details
udf function noop A function that takes a dataframe as input and returns a dataframe as output.
udf_kwargs dict {} The keyword arguments to pass to the udf.
cbs list [] A list of Callbacks to run before and after the mapping. It will add block_counter_cb and msg_bs_cb by default.
verbose bool False Whether to print the batch size.

Usage Example

df = pd.DataFrame(
    {
        "text": [
            ["I worked at Google", "I studied at Harvard"],
            ["I worked at Facebook", "I studied at MIT"],
        ]
    }
)


def ander(df):
    "A simple example of a user defined function that joins the text in each row with ' and '."

    df["joined"] = " and ".join(df["text"])
    return df


m = Mapper(udf=ander)
df = m(df)
